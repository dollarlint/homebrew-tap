# dollarlint Homebrew Tap

Install the latest `dollarlint` release with Homebrew:

```sh
brew install dollarlint/tap/dollarlint
```

Or tap the repository first:

```sh
brew tap dollarlint/tap
brew install dollarlint
```

Formula updates are generated by GoReleaser from
[`dollarlint/dollarlint`](https://github.com/dollarlint/dollarlint).
